# Star Strike

## Simple Space shooting game

Star Strike is a simple space shooting game. Made in Godot. WHere you have to fight against the enemy spaceships and survive agains the space meteors to save your planet.

- Three type of enemies: fast shooter, slow shooter, bounce shooter

- Falling Meteor 

- Three type of powerups: rapid strike, shield, life

- Mobile Playable

- Start with 3 levels, max possible 5 levels

- Slowly enemy get spawned faster.

- Cool bullet effects

- Start screen, instuctions, end screen
